# 线性代数
Numpy为线性代数计算提供了许多功能。 您可以轻松转置矩阵、创建一个单位矩阵、将它们相乘或者计算它们的轨迹。
比如转置矩阵可以写成：
```
a = np.array([1,2,3],[0,1,3])
a
a.T
a.transpose()
```
使用eye函数来创建单位矩阵：
```
np.eye(3)
```
要将矩阵相乘，请使用dot或matmul函数。 请记住，此操作会考虑维度。 如果矩阵b是2 x 2，我们可以看到b x a为：
```
b = np.array([2,1],[3,2])
np.dot(a,b)
```
而当a x b时会报错：
```
np.dot(a,b)
```
